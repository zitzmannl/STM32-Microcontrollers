# Keypad Interface

STM32 keypad interfacing projects involving matrix keypad scanning, GPIO input/output control, and embedded systems logic.

## Topics Covered
- Matrix keypad scanning
- GPIO row/column interfacing
- Embedded input processing
- LED response systems
- Hardware debugging

## Skills Demonstrated
- STM32 microcontroller development
- Embedded C programming
- GPIO interfacing
- Input/output logic design
- Hardware/software integration
